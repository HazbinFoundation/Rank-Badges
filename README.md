## Installation

Install with composer:

```sh
composer require hazbin/rank-badges:"*"
```

or [Extension Manager](https://discuss.flarum.org/d/33955-extension-manager-stable-release-install-flarum-via-zip-file):

```sh
hazbin/rank-badges:"*"
```
## Updating

```sh
composer require hazbin/rank-badges:"*"
php flarum cache:clear
```
